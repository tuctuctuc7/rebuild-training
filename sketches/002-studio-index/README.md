## Variant: Studio Index

### Design stance
A calm editorial portfolio that prioritizes scanability and complete-page usefulness over visual spectacle.

### Key choices
- Layout: sticky side rail, image-led hero, indexed sections
- Typography: large neutral grotesk with restrained orange punctuation
- Interaction: project category filters
- Whole-page system: Work → About → Notes → LinkedIn

### Trade-offs
- Strong at: clarity, trust, project browsing, long-term extensibility
- Weak at: less overtly futuristic or theatrical

### Best for
A durable personal site that should feel like an operator’s studio rather than a campaign landing page.
