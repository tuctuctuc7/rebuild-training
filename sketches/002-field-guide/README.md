## Variant: Field Guide

### Design stance
A warm, playful personal publication built on a strict chapter-based information hierarchy.

### Key choices
- Layout: conventional top nav, chapter sequence, project index, story/principles, notes
- Typography: bold editorial sans without overlaying text on imagery
- Color: warm paper, ink purple, lime/coral/lilac accents used sectionally
- Interaction: chapter links and direct LinkedIn path; the project index stays deliberately stable

### Trade-offs
- Strong at: personality, readability, breadth of content
- Weak at: less conventionally premium; requires disciplined color use

### Best for
A personal site that should feel human, memorable and useful without becoming a UI gimmick.
